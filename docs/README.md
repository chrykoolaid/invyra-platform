# Invyra Platform Documentation

This documentation folder belongs to the `invyra-platform` production platform repository.

It does not document the static website build and does not include Inventory runtime operations.

## Current Documentation Areas

- Architecture boundaries
- Local development
- Security baseline
- Migration workflow
- Future production readiness

## Repository Boundary

`invyra-platform` controls platform access, tenant resolution, licensing, devices, environments, portal runtime, audit, support, and Inventory launch eligibility.

Inventory operations remain in `invyra-inventory`.

Website pages and static previews remain in `invyra-website`.
